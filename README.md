# JSON Stack Viewer

A modern desktop JSON viewer and editor built with Electron and React.

## Features

- ✨ Multi-tab support
- 🔍 Advanced search with highlighting
- 📊 Split-view editor and viewer
- 🎨 Syntax highlighting
- 📁 Collapsible JSON tree
- ⌨️ Keyboard shortcuts
- 💾 Auto-save tabs
- 🎯 Quick tab search (Ctrl+P)

## Installation

### For Users
Download the latest release from the [Releases](https://github.com/YOUR_USERNAME/jsonstackviewer/releases) page.

### For Developers
```bash
# Clone repository
git clone https://github.com/YOUR_USERNAME/jsonstackviewer.git
cd jsonstackviewer

# Install dependencies
npm install

# Run in development
npm run dev    # In one terminal
npm start      # In another terminal

# Build for production
npm run build
npm run dist:win
```

## Keyboard Shortcuts

- `Ctrl+T` - New tab
- `Ctrl+W` - Close tab
- `Ctrl+P` - Quick tab search
- `Ctrl+F` - Search in JSON
- `Ctrl+S` - Save (auto-save enabled)
